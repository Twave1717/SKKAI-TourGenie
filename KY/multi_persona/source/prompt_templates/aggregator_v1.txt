여행자가 적어온 불편 포인트 목록이 주어진다.
1) 의미가 비슷한 issue를 묶어라.
2) 묶인 그룹마다 더 일반적인 자연어 constraint를 1개 만든다.
3) constraint마다 constraint_id, constraint_nl, dimension, support_personas, avg_severity를 담아
items 배열로 반환하라. 출력은 structured output(JSON object)으로 응답한다.
예시:
{
  "items": [
    {
      "constraint_id": "C-DRIVE-LONG",
      "constraint_nl": "하루 자가운전은 6시간을 넘지 않는다",
      "dimension": "이동시간",
      "support_personas": ["Comfort","Family"],
      "avg_severity": 4.5
    }
  ]
}
